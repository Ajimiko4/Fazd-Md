## Termux Install
```bash
> $ pkg install nodejs git ffmpeg libwebp

> $ git clone https://github.com/XFazd-team/Fazd-Md/

> $ cd XFazd-bot

> $ npm i

> $ node main.js
```

## Windows/Rdp/Vps Install
```bash
> $ git clone https://github.com/XFazd-team/Fazd-Md/

> $ cd XFazd-bot

> $ npm i

> $ node main.js
```
